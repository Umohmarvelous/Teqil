## Database Schema (Supabase)
Create the following tables. Use Row Level Security with appropriate policies.

### `users` (extends auth.users)
- `id` uuid (pk)
- `full_name` text (nullable; auto‚Äëgenerate if empty)
- `phone` text unique not null
- `email` text unique not null
- `age` int not null check (age >= 18)
- `role` text not null check (role in ('driver','passenger','park_owner'))
- `driver_id` text unique (generated for drivers)
- `profile_photo` text (url)
- `vehicle_details` text (drivers)
- `park_location` text (drivers)
- `park_name` text (drivers)
- `points_balance` int default 0
- `avg_rating` float
- `created_at` timestamptz default now()

### `trips`
- `id` uuid pk
- `driver_id` uuid references users(id)
- `trip_code` text unique not null
- `origin` text not null
- `destination` text not null
- `start_time` timestamptz not null
- `end_time` timestamptz
- `distance` float
- `status` text not null check (status in ('active','completed'))
- `created_at` timestamptz default now()

### `passengers`
- `id` uuid pk
- `trip_id` uuid references trips(id)
- `user_id` uuid references users(id)
- `destination` text (defaults to trip.destination)
- `dropoff_time` timestamptz
- `status` text not null check (status in ('active','completed'))
- `emergency_contacts` jsonb (array of {name, phone})
- `created_at` timestamptz default now()

### `ratings`
- `id` uuid pk
- `trip_id` uuid references trips(id)
- `rater_id` uuid references users(id)
- `rated_id` uuid references users(id)
- `stars` int not null check (stars between 1 and 5)
- `tags` text[]
- `review` text
- `created_at` timestamptz default now()

### `parks`
- `id` uuid pk
- `name` text not null
- `location` text not null
- `owner_id` uuid references users(id)
- `access_code` text (for park owner verification)
- `created_at` timestamptz default now()
